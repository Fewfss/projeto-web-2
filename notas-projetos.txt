# Contexto do Projeto

- Projeto de API REST com Node.js, Express e MongoDB
- Entidades: Authors, Books, CDs, DVDs
- Autenticação com Google via Passport
- Geração de JWT no callback
- Middleware requireToken protege POST, PUT, DELETE
- GET é público
- Testado com Postman usando token JWT
- Objetivo: transformar em sistema de gerenciamento de livros com catálogos
